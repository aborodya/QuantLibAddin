
QuantLibAddin
<http://www.quantlib.org/quantlibaddin>

QuantLibAddin is an API for QuantLIb which facilitates deployment to end user
environments such as spreadsheets.  QuantLibAddin is a subproject of the
QuantLib open source C++ analytics library.

The mailing list for this project is
<https://lists.sourceforge.net/lists/listinfo/quantlib-users>.

Installation instructions are available at
<http://quantlib.org/reposit/docs/latest/build_overview.html>.

A list of changes for this and previous releases can be browsed at 
<http://quantlib.org/quantlibaddin/history.html>.
